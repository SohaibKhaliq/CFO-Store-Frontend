## 📖 Overview

Next.js e-commerce storefront with PWA support, Redux Toolkit state management, Facebook Pixel tracking, Tawk.to live chat, and Swiper-based product carousels.

## 🚀 Quick Start

```bash
git clone https://github.com/SohaibKhaliq/CFO-Store-Frontend.git
cd CFO-Store-Frontend
npm install
npm run dev
```
